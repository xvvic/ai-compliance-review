## 三、四通保全详图

### 通道一 · 直接锁定

#### 3.1 银行账户与金融线索
- 来源：招投标公告中的付款账户、政府公告
- 评估：零星账户可能非主要结算户
- 操作：先行冻结 + 调查令查询全部账户

#### 3.2 对外投资股权
- 字段：对外投资列表（名称/出资比例/注册资本/状态）
- 评估：存续子公司→按出资比例冻结；已被前序冻结→轮候
- 操作：列明股权申请冻结

#### 3.3 知识产权出质质
- 字段：知识产权出质登记（质权人/登记日期）
- 评估：质押权人优先受偿→核实质押金额→评估剩余价值
- 操作：申请对剩余价值部分保全

#### 3.4 动产与不动产抵押/查封
- 字段：动产抵押/土地抵押登记
- 评估：抵押物市场价值 - 抵押债权 = 剩余可执行价值
- 操作：申请对剩余价值部分查封

### 通道二 · 调查追踪

#### 3.5 银行账户深挖线索
- 曾用名追查 + 招投标付款信息推测开户行
- 操作：律师调查令至人民银行查询全部账户

#### 3.6 地址与动产线索
- 字段：注册地址（含历史变更）+ 电信许可地址 + 医疗器械备案地址
- 评估：地址聚集度→建议实地走访
- 操作：核实动产（设备/库存/车辆）/租赁权益

#### 3.7 土地与在建工程
- 字段：土地出让/转让 + 建设工程许可证/施工许可证
- 评估：需核实权属登记 + 抵押状况
- 操作：调查令调取不动产登记 + 保全应付工程款

#### 3.8 高价值知识产权
- 字段：专利（已授权/未授权）/商标（已注册/申请中）/著作权/软著/域名
- 评估：核心商标/高价值专利→保全限制转让
- 操作：向商标局/国家知识产权局发协助执行通知

#### 3.9 业务资质与经营现金流
- 字段：资质证书 + 行政许可
- 评估：资质不可直接保全，但主营业务现金流是切入点
- 操作：冻结银行账户拦截经营收入 + 追踪应收账款

### 通道三 · 债权拦截

#### 3.10 招投标项目应收账款
- 字段：中标项目（招采单位/金额/日期）
- 评估：按付款节点排序→临到期项目优先
- 操作：向招标方/付款方申请到期债权查封

#### 3.11 诉讼胜诉债权
- 字段：作为原告的胜诉案件 + 作为被告的案件
- 评估：胜诉未回款→向案外人法院申请协助冻结
- 操作：逐一核查执行进展

#### 3.12 经营合同预期收益
- 从资质/招投标推断持续性合同收入
- 操作：调查令调取主要业务合同 + 概括性保全

### 通道四 · 关联穿透

#### 3.13 股东出资责任追究
- 字段：股东信息（认缴/实缴/出资比例）+ 实控人/最终受益人
- 评估：实缴不全→加速到期条件评估
- 操作：追加未出资股东为共同被告/被执行人

#### 3.14 分支机构财产执行
- 字段：分支机构列表
- 评估：可直接执行分公司财产
- 操作：保全申请同步列入分支机构

#### 3.15 实控人及关联企业
- 字段：高管控制企业 + 对外任职
- 评估：人格混同可能性（共用账户/同地址/交叉持股）
- 操作：独立财产调查 + 主张揭开公司面纱

#### 3.16 历史关联方追踪
- 字段：注销/卸任关联方
- 评估：违法注销→追加相关责任主体
- 操作：核查清算程序合规性

## 四、诉讼与执行竞合态势

- 4.1 被告竞合分析：前序保全已锁定的财产范围
- 4.2 原告胜诉债权：未执行回款
- 4.3 信用恶化信号：失信/限消/终本/股权冻结（数量+最近日期+对保全策略的影响）
- 4.4 保全顺位风险评估
