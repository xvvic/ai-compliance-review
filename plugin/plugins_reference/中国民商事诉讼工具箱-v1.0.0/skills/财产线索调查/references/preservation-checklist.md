# 财产保全法律依据必检清单

> 每次生成报告时，必须通过法条检索工具实时拉取下列条文的现行有效版本，禁止凭记忆引用。

## 一类：程序基础

| 法规 | 必检条款 | 引用场景 |
|---|---|---|
| 民事诉讼法（最新修正） | 保全条款（诉前/诉讼保全） | 报告速览+各通道保全建议 |
| 民诉法司法解释 | "保全和先予执行"专章 | 担保金额、裁定期限 |

## 引用标准流程

1. 通过法条检索工具检索法规名+关键词
2. 取现行有效版本，记录实施日期
3. 报告中引用格式："根据《XXX》（YYYY修正）第N条第M款规定，……"
4. 报告末尾列出所有引用法规清单
5. 每次生成报告时重检，避免错引旧版本
