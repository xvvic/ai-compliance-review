# 三层审查结构

> 每一项请求权（或抗辩）的审查，按三层递进展开。任一层不成立则该权利被否定。

## 一、三层

1. **权利已产生**：构成要件是否全部满足
2. **权利未消灭**：是否存在使权利消灭的事由
3. **权利可行使**：是否存在暂时阻止行使的事由

只有上一层成立，才进入下一层。

## 二、每层内部的审查步骤

### 第一层：权利已产生

1. **构成要件逐项审查**：每个要件——先明确含义，再对照案件事实，得出结论。含义不清时进入法律解释流程（参 `legal-interpretation-toolkit.md`）。
2. **产生抗辩**：审查是否存在阻断权利产生的事由（行为能力缺陷、意思表示瑕疵、违法、违背公序良俗等）。

### 第二层：权利未消灭

审查清偿、抵销、提存、免除、解除、撤销等使权利已经消灭的事由。

### 第三层：权利可行使

审查同时履行抗辩、不安抗辩、先诉抗辩、留置、诉讼时效抗辩等使权利暂不能行使的事由。

## 三、抗辩链递进

当被告提出抗辩 → 原告可提反抗辩（消灭被告抗辩的反向事由）→ 被告可再抗辩……如此递进直至无新事由。每一层均按三层结构独立审查。

## 四、否认 vs 抗辩的归位

- **否认**：否定对方主张事实 → 在该构成要件项下直接得出"要件未满足"结论
- **抗辩**：承认对方事实但主张另有阻却/消灭/阻止事由 → 进入对应抗辩层审查

归位决定该防御放在攻防结构的哪个节点、举证责任归谁。

## 五、固定排列

```
A. 第一层：权利已产生
   A.1 构成要件
       A.1.1 要件一
       A.1.2 要件二 ...
   A.2 责任范围（如涉及赔偿）
   A.3 产生抗辩

B. 第二层：权利未消灭
   B.1 消灭事由审查

C. 第三层：权利可行使
   C.1 行使障碍审查
```

确认之诉/形成之诉/非讼程序的审查不必严格走完三层，根据规范基础审查相应要件即可。

## 六、抗辩权本身的审查

抗辩权（如时效、同时履行、不安抗辩等）本身也按三层审查：产生→未消灭→可行使。
