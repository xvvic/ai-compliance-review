# 法律解释方法工具箱

> 当涵摄遇到不确定法律概念（如"重大过失""合理期限""显失公平"）时，按本工具箱展开解释论证。

## 一、四种基本解释方法

按以下顺序逐一考察，不得以某一种"明确"为由跳过其余：

1. **文义解释**：条文用语在法律共同体内的通行含义。确定核心语义与可能外延。
2. **体系解释**：本条与同一编/章/节内条文的逻辑协调。外在体系（位置）+ 内在体系（价值）。
3. **历史解释**：立法说明、草案修订过程中所追求的规整意图。
4. **目的解释**：条文在当下法秩序中所要保护的法益与所要避免的损害。多方法冲突时通常居优。

**辅助方法**：合宪性解释（多种解释可能时选合宪者）、比较法解释（仅作启发，不可直接移植）。

## 二、法律漏洞与续造

先穷尽解释 → 确认漏洞存在 → 选择续造工具：

| 工具 | 适用场景 |
|------|---------|
| 类推适用 | 待决情形与已规整情形利益状况相同，存在公开漏洞 |
| 目的性限缩 | 条文文义涵盖过广，适用违反规范目的 |
| 举重明轻/举轻明重 | 梯度关系中的当然推论 |
| 反面推论 | 条文表达了"仅限于"的限定意图 |

## 三、不确定概念的具体化

不给抽象定义，而是**类型化对照**：

1. 列出既往认定为"是"和"否"的典型情形；
2. 对本案事实与最接近的典型情形逐要素比对；
3. 说明差异在评价上是否具有意义。

## 四、论证纪律

- 偏离文义者承担论证负担；
- 涉类推时须证立：(a)漏洞存在 (b)利益状况相同 (c)三个以上评价要素对照；
- 一般条款不得绕过具体规范的明文条件；
- 价值判断须明示连接点（条文/原则/宪法价值），禁止"显然""不言自明"。
