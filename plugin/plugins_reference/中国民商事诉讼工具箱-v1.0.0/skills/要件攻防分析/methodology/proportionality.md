# 论证深度控制

> 不是所有要件都需要详尽论证。论证深度应与争议程度和法律意义成比例。

## 一、三级深度

- **明显满足的要件**：一句话直接论断（"本案存在书面买卖合同，合同关系成立"）
- **可能有争议的要件**：展开"规范含义→事实对照→结论"的完整论证，篇幅适度
- **核心争议要件**：完整展开法律解释、类案对比、对反对观点的回应

## 二、判断标准

针对每个要件回答四个问题：

1. **事实是否落入条文文义的核心范围？** 是→直接论断；否→展开论证
2. **一般人能否准确把握该要件含义？** 是→直接适用；否→进入法律解释（参 `legal-interpretation-toolkit.md`）
3. **双方对该要件是否有争议？** 无→简短带过；有→完整展开
4. **该要件是否决定整个请求权成败？** 是→必须深入；否→适度即可

## 三、两种论证方式

### 展开论证（争议要件）

```
[规范含义] 《民法典》第X条规定……该要件是指……
[事实对照] 本案中……
[结论] 故该要件满足/不满足。
```

### 直接论断（明显要件）

```
本案中，A与B已签订书面买卖合同，合同关系成立，满足第595条之要件。
```

## 四、混合使用

合格的攻防分析必然混合使用——核心争议展开论证，非核心要件直接论断。两种错误：

- 过度展开：每个要件都长篇大论 → 冗余，淹没重点
- 过度简略：核心争议也一句话带过 → 论证不充分

## 五、对反对观点的回应

- 次要观点：一句话回应（"或有观点认为……此说与规范目的相悖，不采"）
- 主要观点：展开"前提—理由—我方反驳"的完整结构
- 判例支持的观点：必须展开并说明与本案的事实差异

## 六、自检

- 每个核心要件是否按争议程度分级？
- 论证深度是否与其重要性匹配？
- 是否避免了全篇展开或全篇简略的极端？
- 是否避免了"显然""不言自明"等回避性表述？
