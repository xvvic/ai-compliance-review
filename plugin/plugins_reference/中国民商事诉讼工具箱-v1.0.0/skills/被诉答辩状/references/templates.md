# 答辩状格式模板

## 一、民事答辩状（一审）

```
民 事 答 辩 状

答辩人（被告）：[姓名/名称]，[基本信息]。
委托诉讼代理人：[姓名]，[律师事务所]律师。

答辩人因原告[姓名/名称]诉答辩人[案由]纠纷一案，现提出答辩意见如下：

一、[答辩意见第一点]
[具体论述：回应原告主张→提出反驳理由→引述证据支撑→得出结论]

二、[答辩意见第二点]
[具体论述]

三、[答辩意见第三点]
[具体论述]

综上所述，原告的诉讼请求缺乏事实和法律依据，请求贵院依法驳回原告的全部诉讼请求。

此致
[法院全称]

答辩人：[签名/盖章]
[年] 年 [月] 月 [日] 日
```

## 二、上诉答辩状（二审被上诉人）

```
民 事 答 辩 状

答辩人（被上诉人，原审[身份]）：[姓名/名称]，[基本信息]。

答辩人因上诉人[姓名/名称]不服[法院全称]（[年]）[字]第[号]号民事判决提起上诉一案，现提出答辩意见如下：

一、一审判决认定事实清楚
[具体论述]

二、一审判决适用法律正确
[具体论述]

三、上诉人的上诉理由不能成立
[逐一回应上诉理由]

综上所述，一审判决认定事实清楚，适用法律正确，请求贵院驳回上诉，维持原判。

此致
[二审法院全称]

答辩人：[签名/盖章]
[年] 年 [月] 月 [日] 日
```

## 三、再审答辩状

```
民 事 答 辩 状

答辩人（被申请人，原审[身份]）：[姓名/名称]，[基本信息]。

答辩人因再审申请人[姓名/名称]申请再审一案，现提出答辩意见如下：

一、再审申请人主张的再审事由不成立
[逐一回应再审申请书中的再审事由]

二、原判决认定事实正确
[具体论述]

三、原判决适用法律正确
[具体论述]

综上所述，再审申请人的再审申请不符合法定再审条件，请求贵院依法驳回再审申请。

此致
[受理再审法院全称]

答辩人：[签名/盖章]
[年] 年 [月] 月 [日] 日
```

## 四、各部分填写指引

### 当事人信息
- 格式同起诉状要求
- 答辩人身份标明"被告"或"被上诉人"或"被申请人"
- 如有委托代理人，应列明

### 答辩意见结构
答辩意见的每一点应遵循"回应+辩驳+结论"三段式：
1. **回应**：明确指出回应原告/上诉人的哪一项主张
2. **辩驳**：从事实和法律两个层面展开反驳
3. **结论**：得出该项主张不成立的结论

### 答辩策略层次
按以下层次组织答辩，优先使用高层次抗辩：
1. **程序抗辩**（最优先）：管辖异议、主体不适格、诉讼时效等
2. **实体抗辩**：否认请求权基础要件成立、提出权利障碍/消灭/阻止抗辩
3. **证据抗辩**：否认对方证据的三性、举证责任分配

### 论证要求
- 每个答辩点必须有事实依据+法律依据
- 引用证据时标明证据编号
- 引用法条时写明条款号
- 避免情绪化表述，保持客观理性

### 格式规范
- 标题：小一号宋体加粗居中
- 正文：小三号仿宋
- 行距：28磅固定值
- 页边距：上下3.7cm，左右2.8cm
- A4纸张
