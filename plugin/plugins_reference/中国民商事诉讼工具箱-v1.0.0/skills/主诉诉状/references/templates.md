# 诉状格式模板

## 一、民事起诉状（一审·自然人原告）

```
民 事 起 诉 状

原告：[姓名]，[性别]，[民族]，[出生日期]，住[住址]，公民身份号码[号码]。
联系电话：[电话]。

被告：[姓名/名称]，[性别/法定代表人]，[民族/职务]，住[住址/住所地]。
联系电话：[电话]。

诉讼请求：
一、[具体请求1]；
二、[具体请求2]；
三、本案诉讼费用由被告承担。

事实与理由：
[按时间顺序叙述案件事实，分段落，每段围绕一个核心事实展开。最后一段简述法律依据和请求基础。]

综上所述，原告依据《中华人民共和国民法典》第[条]条之规定，向贵院提起诉讼，请求判如所请。

此致
[法院全称]

具状人：[签名]
[年] 年 [月] 月 [日] 日

附：1. 本状副本[X]份；
    2. 证据材料[X]份。
```

## 三、民事上诉状（二审）

```
民 事 上 诉 状

上诉人（原审[原告/被告]）：[姓名/名称]，[基本信息同上]。
被上诉人（原审[被告/原告]）：[姓名/名称]，[基本信息同上]。

上诉人因与被上诉人[案由]纠纷一案，不服[法院全称]（[年]）[字]第[号]号民事判决，现提起上诉。

上诉请求：
一、撤销[法院简称]（[年]）[字]第[号]号民事判决[第X项]；
二、[改判请求/发回重审请求]；
三、本案一、二审诉讼费用由被上诉人承担。

上诉理由：

一、[原判认定事实错误/适用法律错误/违反法定程序]
[具体论述]

二、[第二个上诉理由]
[具体论述]

此致
[二审法院全称]

上诉人：[签名/盖章]
[年] 年 [月] 月 [日] 日
```

## 四、再审申请书

```
民 事 再 审 申 请 书

再审申请人（原审[身份]）：[姓名/名称]，[基本信息]。
被申请人（原审[身份]）：[姓名/名称]，[基本信息]。

再审申请人因与被申请人[案由]纠纷一案，不服[法院全称]（[年]）[字]第[号]号民事判决/裁定，依据《中华人民共和国民事诉讼法》第二百零七条第[项]项之规定，向贵院申请再审。

再审请求：
一、撤销[判决/裁定]；
二、[改判请求]；
三、本案诉讼费用由被申请人承担。

事实与理由：

一、[再审事由对应的具体事实和理由]
[具体论述，对应民诉法第207条各项]

此致
[上一级法院全称]

再审申请人：[签名/盖章]
[年] 年 [月] 月 [日] 日
```

## 五、各部分填写指引

### 当事人信息
- 自然人：姓名、性别、民族、出生日期、住址、身份证号、联系电话
- 法人：全称、住所地、统一社会信用代码、法定代表人姓名及职务、联系电话
- 多被告时逐一列明，按主次排序

### 诉讼请求
- 具体、明确、可执行
- 金钱给付：写明具体金额+利息计算方式+起止日期
- 行为给付：写明具体行为内容
- 确认请求：写明确认对象
- 末项固定为"本案诉讼费用由被告承担"

### 事实与理由
- 按时间顺序叙述核心事实
- 区分"无争议事实"和"争议事实"
- 引述关键证据（但不展开论证）
- 最后一段点出法律依据

### 格式规范
- 标题：小一号宋体加粗居中
- 正文：小三号仿宋
- 行距：28磅固定值
- 页边距：上下3.7cm，左右2.8cm
- A4纸张
