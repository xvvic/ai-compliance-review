# 法律图表构图模式

本文件为诉讼可视化skill的配套参考，提供常见法律图表的构图模式和设计要点。

## 1. 法律关系图

### 布局原则
- 原告左、被告右
- 时间从左到右
- 层级从上到下（母公司上、子公司下）
- 连接最多的实体放中心
- 同一实体全图只出现一次
- 同类实体同形状、同颜色

### 线条语义
- 实线 = 已生效合同/确定关系
- 虚线 = 争议中/主张中的关系
- 粗线 = 核心争议
- 箭头方向 = 权利义务/资金/物权流向
- 线条粗细反映金额比重

### 框架语义
- 虚线框 = 利益共同体（母子公司、关联方）
- 实线框 = 确定归属分组
- 框的位置和范围本身表达观点

### 避免事项
- 线条交叉：重排节点位置而非绕线
- 线条长途奔袭：把需要连接的节点放近
- 同一实体重复出现

## 2. 时间轴线图

### 基本结构
横向或纵向时间轴，标注关键时间和事件。

### 三种进阶布局

**纵向划分（最常用）**：利用时间轴上下空间表达对比关系。
- 甲方行为 vs 乙方行为
- 合同约定 vs 实际履行
- 原告行为 vs 被告行为
- 应付金额 vs 实付金额

**横向划分**：用括号或色块表达时间区间。
- 保证期间 / 诉讼时效 / 合同履行期 / 工期计划 vs 实际工期

**行为分层**：纵向空间分为 3-4 层。
- 程序分层：一审/二审/再审/执行
- 多方行为分层
- 复杂工期分层

### 设计规则
- 不超过 12 个主要节点
- 密集时间线拆为：摘要时间线 + 事件表格 + 分析页
- 时间节点用小圆点标注
- 事件框控制在 2-4 行

## 3. 判决书三件套

长判决书不应产出一张过载图表，默认拆为三张：

**图一：当事人关系图** — 当事人、诉讼地位、合同/担保/保全关系

**图二：裁判逻辑图（争点树）** — 争议焦点 → 法院审理路径 → 认定结论 → 最终判决

**图三：金额与责任分配图** — 起始金额 → 扣减 → 最终金额 → 利息 → 责任承担

## 4. 建设工程纠纷

### 基本结构图模式
- 合法分包：发包人 → 总包 → 专业分包/劳务分包
- 整体转包：发包人 → 总包 ···→ 实际施工方（虚线表示名义关系）
- 挂靠：虚线表示挂靠关系，实线表示实际施工关系

### 三层泳道布局
- 上层：合同与监理关系
- 中层：付款断裂/违约触发
- 下层：索赔、停工、追偿、担保执行

### 分图规则
- 超过 6 个参与方或 8 条关系线时拆图
- 拆法：合同链图 + 索赔路径图

## 5. 票据纠纷

### 三线分离法
- 蓝色线 = 票面记载流转
- 灰色虚线 = 实际流转
- 红色线 = 资金流向（与票面方向相反）

三线对比揭示纠纷本质。

## 6. 公司纠纷

### 股权结构图
- 股东少于5人：表格
- 股权比例对比：饼图/环形图
- 多层级结构：树形图

### 股权变动
- 单次变动：左/上=变动前，右/下=变动后
- 多次变动：同一实体全程同色，色块面积与持股比例对应

### 关键规则
- 股东在上、子公司在下
- 持股比例标注在连接线上
- 虚线分隔不同法域

## 7. 诉讼策略图

### 推荐结构
- 中心：目标争议/主诉请
- 左侧：原告进攻路径
- 右侧：被告防守路径
- 底部：证据链与薄弱点
- 顶部：诉讼目标与风险提示

## 8. 节点文本规则

**好**：2-4 行精炼标签
```
总包方
华夏一局
分包幕墙工程：1.5亿
```

**坏**：整段文字塞进方框

长描述用旁注或脚注承载。

## 9. 连线规则

- 关系标签放在连线中部，不放在节点上方
- 标签加白色背景
- 虚线红色 = 争议/断裂的付款流
- 紫色线 = 担保/保全/追偿
- 灰色虚线 = 监督/背景关联
- 避免同两节点间多条平行线

## 10. 色彩使用原则

- 低饱和度商务色
- 颜色目的：区分类别、突出观点、引导注意力
- 同一实体全图同色
- 主色不超过 3-4 种
- 用色彩表达观点：红色=违约，绿色=适当履行，灰色=背景事实

## 11. 案由→图表选型建议

| 案由类型 | 推荐首选图表 | 辅助图表 |
|---------|------------|---------|
| 合同纠纷 | 法律关系图 | 时间轴（履约对比） |
| 侵权纠纷 | 时间轴（行为序列） | 因果关系图 |
| 公司纠纷 | 股权结构图 | 股权变动时间轴 |
| 建设工程 | 三层泳道图 | 工期对比时间轴 |
| 票据纠纷 | 三线分离图 | 时间轴 |
| 借贷纠纷 | 法律关系图 | 借还款数据表 |
| 劳动争议 | 时间轴 | 法律关系图 |
| 知识产权 | 权利链图 | 侵权行为时间轴 |
