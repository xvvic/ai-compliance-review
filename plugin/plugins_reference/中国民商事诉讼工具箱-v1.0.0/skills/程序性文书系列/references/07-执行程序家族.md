## ① 申请执行书

适用：生效法律文书（判决、裁定、调解书、仲裁裁决、公证债权文书等）确定的义务，义务人拒不履行，权利人向有管辖权的法院申请强制执行。

请求事项写法：
```
请求依据……〔（××××）×××民初×号民事判决书/调解书/×仲裁裁决书〕，对被执行人×××强制执行，实现下列内容：
一、给付申请执行人……元及迟延履行期间的债务利息（加倍部分债务利息按民诉法第264条计算）；
二、由被执行人承担本案执行费用。
```
专属要件：①写明生效法律文书文号、生效日期、确定的义务；②**申请执行期间二年**，从履行期间最后一日起算（分期履行从每次期间最后一日起算）；③管辖法院为一审法院或被执行财产所在地同级法院；④可一并申请加倍迟延履行利息、罚息。
法律依据：民诉法第235条、第246条、第264条。
附：生效法律文书副本、生效证明、申请执行人主体材料、被执行人财产线索（可衔接 `/财产线索调查`）。

## ② 恢复执行申请书

适用：案件经"终结本次执行程序"（终本）后，申请执行人发现被执行人有可供执行的财产或财产线索，申请恢复执行。

请求事项写法：
```
本案〔（××××）×××执×号〕已于××××年××月××日裁定终结本次执行程序，现申请执行人发现被执行人×××有下列可供执行财产/线索：……，请求恢复执行。
```
专属要件：①提供具体财产线索（银行账户、不动产、车辆、股权、到期债权等）；②**恢复执行不受申请执行时效期间限制**；③可同步申请查控措施。
法律依据：《终本规定》第9条。
附：财产线索证据（可衔接 `/财产线索调查`）。

## ③ 限制高消费申请书

适用：被执行人未按执行通知履行义务，申请对其采取限制高消费及有关消费措施（限制乘坐飞机/高铁、星级酒店、购房、子女就读高收费私立学校等）。

请求事项写法：
```
被执行人×××未按执行通知履行〔（××××）×××执×号〕确定的义务，请求依法对其采取限制消费措施；被执行人为单位的，请求对其法定代表人/主要负责人/影响债务履行的直接责任人员等采取限制消费措施。
```
专属要件：①被执行人未按执行通知履行；②单位被执行人的，限消对象及于法定代表人、主要负责人、实际控制人、直接责任人员；③说明拟限制的消费行为类型。
法律依据：《限制高消费规定》（法释〔2015〕17号）第1、3条。

## ④ 将被执行人列入失信名单申请书

适用：被执行人有履行能力而拒不履行，或存在伪造证据、规避执行、违反财产报告/限消令等失信情形，申请将其纳入失信被执行人名单。

请求事项写法：
```
被执行人×××具有下列情形：……（有履行能力而拒不履行/以伪造证据、暴力威胁等妨碍执行/规避执行/违反财产报告制度/违反限制消费令/无正当理由拒不履行执行和解协议等），请求依法将其纳入失信被执行人名单。
```
专属要件：①对照《失信名单规定》第1条所列六类情形逐项论证；②提供相应证据。
法律依据：《失信名单规定》（法释〔2017〕7号修正）第1条。
> 注意：单纯"暂无财产可供执行"不属失信情形，避免误用。

## ⑤ 追加/变更被执行人申请书

适用：执行依据生效后，因主体合并分立、出资瑕疵、抽逃出资、一人公司财产混同、保证人、遗产继承、企业法人分支机构等法定情形，申请变更或追加被执行主体。

请求事项写法：
```
请求依法追加/变更×××（基本信息）为本案〔（××××）×××执×号〕被执行人，在……范围内（如未出资本息范围、抽逃出资范围、遗产范围内）承担责任。
```
专属要件：①追加/变更须有法定情形与明确法律依据（《变更追加规定》对应条文）；②写明承担责任的范围与限额；③区分"应当追加"与"可以申请并经审查"的情形；④被追加方对追加裁定不服可提执行异议/异议之诉。
法律依据：《变更、追加当事人规定》（法释〔2016〕21号，2020修正）相应条文（按情形核验具体条号）。
附：证明法定情形的材料（工商档案、验资报告、继承材料、保证合同等）。

## ⑥ 执行异议书（多场景）

适用：分两类，提交前先判明类型——

**场景A · 执行行为异议（程序异议，民诉法第236条）**
- 主体：当事人、利害关系人
- 对象：认为**执行行为**（查封、扣押、冻结、评估、拍卖、分配等程序行为）违反法律规定
- 请求写法：`请求撤销/纠正……执行行为（如撤销对×××财产的查封）。`
- 救济：法院15日内审查裁定；不服的，自裁定送达之日起**10日内**向**上一级法院申请复议**。

**场景B · 案外人执行标的异议（实体异议，民诉法第238条）**
- 主体：案外人
- 对象：对**执行标的**主张所有权或足以排除强制执行的实体权利
- 请求写法：`请求中止/停止对×××标的（如登记在被执行人名下但实际归案外人所有的×××）的执行。`
- 救济：法院15日内审查；理由成立裁定中止执行，不成立裁定驳回。案外人对驳回裁定不服，认为原判决/裁定错误的，依审判监督程序办理；**与原判决/裁定无关的，自裁定送达之日起15日内提起执行异议之诉**（转 cat7「执行异议之诉起诉状」）。

通用专属要件：①明确区分场景A/B（程序 vs 实体；当事人/利害关系人 vs 案外人；复议 vs 诉讼）；②场景B须举证排除执行的实体权利（所有权、未过户买受人物权期待权、租赁权等）；③异议审查期间一般不停止执行（提供担保可请求停止处分）。
法律依据：民诉法第236条、第238条；《执行程序若干问题的解释》（法释〔2020〕21号，内部旧条号须换算为236/237/238）；《执行异议和复议案件若干问题的规定》（法释〔2015〕10号，2020修正）。

## 通用附件与提交提示
- 份数：正本交执行/受理法院，按相对方人数备副本。
- 时限红线：执行行为异议复议 10 日；案外人异议之诉 15 日；申请执行期间 2 年。
- 后续衔接：保全/查控到期续行须自行登记到期日并及时办理（冻结存款 1 年、查封动产 2 年、查封不动产 3 年，到期前续封）；异议被驳回拟提诉的转本系列 `08-其他程序类`（执行异议之诉）；财产线索补强转 `/财产线索调查`。
