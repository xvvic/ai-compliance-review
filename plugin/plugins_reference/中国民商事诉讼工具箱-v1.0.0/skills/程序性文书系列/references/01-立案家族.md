# 06 · 立案家族

> 配合 `00-通用规范.md` 使用。覆盖 6 种立案/主体资格文书：①法定代表人身份证明书（法人）②主要负责人身份证明书（其他组织）③共同诉讼代表人推选书 ④授权委托书（公民）⑤授权委托书（法人/其他组织）⑥推荐函（公民代理）。
> 法条基准：民诉法 2023 修正 + 解释（已按现行编号核验）。
>
> **定位**：本家族采用最高人民法院文书样式书的**通用范本**（公民版/法人版），不绑定特定主体。用户可基于此范本自行调整为团队定制版。

## 现行法条速查（须经核验闸门复核）

| 文书 | 现行依据 |
|------|------|
| 法定代表人/主要负责人身份证明书 | 民诉法第51条（法人由法定代表人、其他组织由主要负责人进行诉讼）；解释第50条（法定代表人以登记为准；变更未登记的处理） |
| 共同诉讼代表人推选书 | 民诉法第56条、第57条（人数众多可推选代表人；变更/放弃/承认诉请、和解须经被代表人同意） |
| 授权委托书 | 民诉法第61条（委托代理人范围，可委托1–2人）、第62条（须提交授权委托书，记明委托事项与权限；特别授权事项） |
| 推荐函（公民代理） | 民诉法第61条第二款第三项；解释第87条（社会团体推荐条件）、第88条第五项（社区/单位推荐公民应提交身份证件、推荐材料） |

## ① 法定代表人身份证明书（法人）

适用：法人作为当事人，证明法定代表人身份以代表法人进行诉讼。

要素：
```
法定代表人身份证明书

×××（法人全称），住所地……，统一社会信用代码……。
×××，性别，××××年××月××日出生，系本单位法定代表人，职务……。
特此证明。
（法人盖章）
××××年××月××日
```
专属要件：①法定代表人**以依法登记的为准**；②已变更但未完成登记、变更后的法定代表人要求代表法人参加诉讼的，法院应予准许（解释第50条）；③加盖法人公章。
法律依据：民诉法第51条；解释第50条。

## ② 主要负责人身份证明书（其他组织）

适用：不具备法人资格的"其他组织"（个人独资企业、合伙企业、个体工商户、法人分支机构等）作为当事人，证明主要负责人身份。

要素：与①类似，将"法定代表人"改为"主要负责人"，注明其他组织的类型、字号/名称、经营场所、负责人职务。
专属要件：①确认属民诉法上"其他组织"范畴；②由其主要负责人进行诉讼；③加盖该组织印章或负责人签字。
法律依据：民诉法第51条；解释第52条（其他组织范围，须核验）。

## ③ 共同诉讼代表人推选书

适用：一方当事人人数众多的共同诉讼，由当事人推选1–5名代表人进行诉讼。

要素：
```
推选书

我们系……（案由）一案的共同原告/被告，现共同推选×××（基本信息）为本案诉讼代表人，代表全体推选人参加诉讼。
代表人变更、放弃诉讼请求或者承认对方诉讼请求、进行和解的，须经被代表的当事人同意。
推选人（逐一签名/盖章）：……
××××年××月××日
```
专属要件：①列明全体推选人并逐一签名；②**代表人作出变更/放弃诉请、承认对方诉请、和解等处分行为，须经被代表人同意**；③明确代表人权限范围。
法律依据：民诉法第56条、第57条。

## ④ 授权委托书（公民）

适用：自然人当事人委托诉讼代理人。

要素：
```
授权委托书

委托人：×××（基本信息）
受托人：×××（基本信息/律师及所属律所）
现委托上述受托人代理本人与×××……（案由）一案的诉讼事宜。
代理权限：□一般授权（代为参加诉讼、提交材料等）
　　　　　□特别授权（代为承认、放弃、变更诉讼请求，进行和解，提起反诉或者上诉，代为签收法律文书等——须逐项列明）
委托期限：……
委托人（签名/捺印）：
××××年××月××日
```
专属要件：①**记明委托事项和权限**；②代为承认/放弃/变更诉请、和解、提起反诉或上诉等属**特别授权**，必须在授权委托书中明确列明，笼统写"全权代理"不发生特别授权效力；③可委托1–2人。
法律依据：民诉法第61条、第62条。

## ⑤ 授权委托书（法人/其他组织）

适用：法人或其他组织委托诉讼代理人。

要素：在④基础上，委托人栏填法人/其他组织全称并由**法定代表人/主要负责人**署名、加盖单位公章；同时提交法定代表人/主要负责人身份证明书（①或②）。
专属要件：①单位公章 + 法定代表人签名；②权限同④，特别授权须逐项列明；③与身份证明书配套提交。
法律依据：民诉法第51条、第61条、第62条。

## ⑥ 推荐函（公民代理）

适用：非律师的公民（当事人所在社区、单位或有关社会团体推荐的人员）担任诉讼代理人时，由推荐主体出具推荐函。

要素：
```
推荐函

兹推荐本……（社区/单位/社会团体名称）的×××（基本信息、职务/与本组织关系）担任×××与×××……（案由）一案的诉讼代理人。
被推荐人系本组织负责人/与本组织有合法劳动人事关系的工作人员/本组织成员。
（推荐主体盖章）
××××年××月××日
```
专属要件：①**社会团体推荐**须满足解释第87条四项条件（非营利性法人、被代理人为其成员或当事人住所地在其活动地域、代理事务属章程业务范围、被推荐人为其负责人或职工）；②**社区/单位推荐**须提交身份证件与推荐材料（解释第88条第五项）；③专利代理人经全国专利代理人协会推荐可在专利纠纷案件代理。
法律依据：民诉法第61条第二款第三项；解释第87条、第88条。

## 通用附件与提交提示
- 配套材料（解释第88条）：律师提交执业证 + 律所证明；基层法律服务工作者提交执业证 + 介绍信 + 辖区证明；近亲属提交身份证件 + 亲属关系证明；工作人员提交身份证件 + 劳动人事关系证明；社区/单位推荐公民提交身份证件 + 推荐材料。
- 份数：原件交法院备案，副本随诉讼材料一并提交。
- 后续衔接：立案材料齐备后转 `/主诉诉状`；立案后自行登记答辩期/举证期/上诉期等关键期限并提醒用户核对。
