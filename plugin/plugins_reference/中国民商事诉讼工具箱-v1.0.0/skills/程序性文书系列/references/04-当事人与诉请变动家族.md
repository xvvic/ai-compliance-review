# 03 · 当事人与诉请变动家族

> 配合 `00-通用规范.md` 使用。覆盖 5 种文书：①增加/变更/放弃诉讼请求申请书 ②追加必要共同诉讼当事人申请书 ③无独立请求权第三人参加诉讼申请书 ④撤回起诉申请书 ⑤撤回反诉申请书。
> 法条基准：民诉法 2023 修正（已按现行编号核验）。

## 现行法条速查（须经核验闸门复核）

| 制度 | 民诉法 2023 | 解释（法释〔2022〕11号） |
|------|------------|------|
| 放弃/变更诉讼请求、反诉 | 第54条（原告可放弃或变更诉请；被告可承认/反驳、提起反诉） | 第232条（增加诉请、反诉、第三人请求应在举证期限届满前提出，合并审理） |
| 共同诉讼 | 第55条 | 第73–74条（必要共同诉讼追加） |
| 通知追加必要共同诉讼当事人 | 第135条（必须共同进行诉讼的当事人未参加的，法院应通知其参加） | 第73–74条 |
| 第三人参加诉讼 | 第59条（有独立请求权/无独立请求权第三人） | 第81–84条 |
| 撤回起诉 | 第148条（宣判前原告申请撤诉，准许与否由法院裁定） | 第238条（撤诉后同一诉请再诉应受理）、第239条 |
| 撤回反诉 | 第54条（反诉）+ 第148条（撤诉规则准用） | 第238–239条 |

## ① 增加/变更/放弃诉讼请求申请书

适用：原告（或反诉原告）在诉讼中增加、变更诉讼请求，或部分/全部放弃诉讼请求。

请求事项写法（按情形择一/组合）：
```
一、增加诉讼请求：请求判令被告……（新增请求，如增加利息/违约金/数额）；
二、变更诉讼请求：将原第×项诉讼请求"……"变更为"……"；
三、放弃诉讼请求：放弃原第×项诉讼请求"……"。
```
专属要件：①增加/变更诉请应在**举证期限届满前**提出（解释第232条），逾期法院可不组织质证或要求说明理由；②放弃诉请是当事人处分权，但不得损害国家/集体/他人合法权益；③变更诉请可能影响管辖与举证，需评估。
事实和理由：写明案号案由、变动的事实依据（如新证据、计算更正、和解部分达成）。
法律依据：民诉法第54条；解释第232条。

## ② 追加必要共同诉讼当事人申请书

适用：必须共同进行诉讼的当事人未参加诉讼（诉讼标的共同，如共有人、共同侵权人、连带责任人、合伙人），申请法院通知其作为共同原告/被告参加。

请求事项写法：
```
请求依法追加×××（基本信息）作为本案共同被告/共同原告参加诉讼。
```
专属要件：①论证拟追加方与本案诉讼标的的**共同性**（共有、连带、共同行为）；②说明属"必须共同进行诉讼"而非普通共同诉讼；③追加被告由法院裁定，追加为共同原告须征得其同意，不同意的可不列为原告（解释第73–74条）。
法律依据：民诉法第55条、第135条；解释第73–74条。
附：证明共同权利义务关系的材料（共有证明、合同、连带责任依据等）。

## ③ 无独立请求权第三人参加诉讼申请书

适用：案件处理结果与某方有法律上的利害关系，该方申请作为无独立请求权第三人参加诉讼（亦可由当事人申请通知其参加）。

请求事项写法：
```
请求准许×××（基本信息）作为无独立请求权第三人参加本案诉讼。
（如系当事人申请通知：请求通知×××作为无独立请求权第三人参加诉讼。）
```
专属要件：①论证与案件处理结果的**法律上利害关系**（非事实上、经济上的间接关联）；②区分有独立请求权第三人（以本诉双方为被告另行主张）与无独立请求权第三人（辅助一方）；③第三人可能被判承担民事责任的，享有当事人诉讼权利义务。
法律依据：民诉法第59条；解释第81–84条。
附：证明利害关系的材料。

## ④ 撤回起诉申请书

适用：原告在法院**宣判前**申请撤回起诉。

请求事项写法：
```
申请人（原告）×××诉被申请人（被告）×××……（案由）一案〔（××××）×××民初×号〕，现申请人申请撤回起诉。
```
专属要件：①须在宣判前提出；②是否准许由法院裁定（撤诉损害国家/集体/他人利益的，法院不予准许）；③说明撤诉原因（如和解、另寻救济、暂缓主张）。
法律依据：民诉法第148条；解释第238–239条。
> 风险提示：①一审撤诉后以同一诉请再起诉，法院应受理；②按撤诉处理与裁定撤诉的诉讼时效、案件受理费退还（一般减半退还）后果不同；③上诉期内撤回上诉与撤回起诉不同。撤诉前评估诉讼时效与再诉可行性（自行核算时效并提醒用户核对）。

## ⑤ 撤回反诉申请书

适用：反诉原告（本诉被告）申请撤回已提起的反诉。反诉是独立之诉，撤回反诉准用撤诉规则。

请求事项写法：
```
本案〔（××××）×××民初×号〕中，申请人（反诉原告/本诉被告）×××已提起反诉，现申请撤回该反诉。
```
专属要件：①宣判前提出；②是否准许由法院裁定；③撤回反诉不影响本诉审理；④注明反诉受理费处理。
法律依据：民诉法第54条、第148条；解释第238–239条。

## 通用附件与提交提示
- 份数：正本交法院，按对方当事人数备副本；追加当事人/第三人参诉的，另备拟追加方副本。
- 时限红线：增加/变更诉请、反诉应在举证期限届满前提出；撤诉/撤反诉须宣判前提出。
- 后续衔接：诉请或当事人变动后同步更新 `/主诉诉状`、`/要件攻防分析`、`/证据目录`；撤诉后自行跟踪再诉时效并提醒用户核对。
