# 07 · 其他程序类（高论证深度）

> 配合 `00-通用规范.md` 使用。覆盖 4 种**非纯填空、需较高论证深度**的程序文书：①实现担保物权程序申请书 ②撤销仲裁裁决申请书 ③第三人撤销之诉起诉状 ④执行异议之诉起诉状。
> 法条基准：民诉法 2023 修正 + 仲裁法 + 相关解释（条号须经核验闸门复核）。
>
> **⚠️ 与前六类不同——本家族强制要求**：
> 1. **充分收集信息与文件**：当事人须提供完整事实材料、证据、相关生效文书/合同/裁决书原件复印件。
> 2. **充分论证**：除程序特殊要件外，实体论证（权利存在、损害事实、因果、排除执行的权利等）必须充分展开，不得套模板。
> 3. **充分调用 MCP**：写作前通过法律检索后端（元典/北大法宝，运行时自动适配+灾备，见 qoder.md「外部能力后端」）校验全部法条与裁判规则，失败标注`[L4-法条待验证]`；案例支撑通过法律检索后端的案例检索获取；如用户已配置诉讼知识库（qmind），可检索相关评注资料辅助论证。
> 4. 这四种文书更接近"实体诉讼文书"，可与 `/主诉诉状`、`/要件攻防分析` 协同。

## ① 实现担保物权程序申请书

适用：担保物权人（抵押权人、质权人、留置权人）或其他有权请求实现担保物权的人，在债务人不履行到期债务或发生当事人约定的实现担保物权情形时，**不经诉讼**直接申请法院裁定拍卖、变卖担保财产。

管辖：担保财产所在地或担保物权登记地基层人民法院。

请求事项写法：
```
请求依法裁定准许拍卖/变卖被申请人×××提供的担保财产（具体财产：……，担保登记××），以所得价款优先清偿申请人享有的债权……元及利息。
```
论证重点（须充分）：①**担保物权合法有效设立**（合同有效、登记/交付完成、主债权真实）；②**实现条件成就**（债务到期未清偿，或约定/法定实现事由发生）；③担保财产范围、债权数额、优先受偿顺位的具体认定；④是否存在权属争议或实体争议——若被申请人提出实质性异议致案件转为民事权益争议，法院将裁定终结特别程序、告知另行起诉（第186条）。
法律依据：民诉法第184条、第203–204条；民法典担保物权相关条文。
附：主债权合同、担保合同、担保物权登记证明、债权数额计算、催收凭证。

## ② 撤销仲裁裁决申请书

适用：当事人有证据证明仲裁裁决存在法定撤销事由，向**仲裁委员会所在地中级人民法院**申请撤销裁决。须在收到裁决书之日起**6个月内**提出。

请求事项写法：
```
请求依法撤销×××仲裁委员会作出的（××××）×仲字第×号仲裁裁决。
```
论证重点（须充分，逐项对照仲裁法第58条）：①没有仲裁协议；②裁决事项不属仲裁协议范围或仲裁委无权仲裁；③仲裁庭组成或仲裁程序违反法定程序；④裁决所根据的证据是伪造的；⑤对方当事人隐瞒了足以影响公正裁决的证据；⑥仲裁员索贿受贿、徇私舞弊、枉法裁决；以及法院认定裁决**违背社会公共利益**的。每一主张须有证据与具体事实支撑，不得泛泛而论。
法律依据：仲裁法第58–61条。
> 区分提示：①**撤销仲裁裁决**（仲裁法第58条，向中院申请撤销）与**不予执行仲裁裁决**（执行程序中提出，民诉法第244条）路径不同，事由相近但程序、时点、后果有别——勿混淆，必要时在易混淆消歧中向用户澄清意图；②劳动争议仲裁、涉外仲裁的撤销规则另有规定。
附：仲裁裁决书、仲裁协议、证明撤销事由的证据。

## ③ 第三人撤销之诉起诉状

适用：未参加他人之间已生效诉讼的第三人，因**不能归责于本人的事由**未参加诉讼，有证据证明生效判决、裁定、调解书的**部分或全部内容错误，损害其民事权益**，自知道或应当知道之日起**6个月内**，向**作出该生效裁判的法院**提起撤销之诉。

诉讼请求写法：
```
一、请求撤销×××人民法院（××××）×××民×字第×号民事判决书/裁定书/调解书中……部分（或全部）；
二、（如有）请求改判/确认……
```
论证重点（须充分）：①原告**适格**——属有独立请求权或无独立请求权第三人，且对原案诉讼标的有法律上利害关系；②**非因本人原因未参加原诉讼**（如未被通知、被恶意串通排除）；③原生效裁判内容**错误且损害其民事权益**，须具体指明错误之处与损害；④**6个月起诉期间**的起算与遵守；⑤举证证明上述要件。
法律依据：民诉法第59条第3款；解释第292–303条。
> 关联：若涉嫌虚假诉讼/恶意串通损害第三人利益，可结合民诉法第115条。
附：证明利害关系、未参加诉讼无过错、原裁判错误及损害的全部证据。

## ④ 执行异议之诉起诉状

适用：执行程序中，案外人或申请执行人对执行标的提出异议，经法院裁定后不服，且异议与原判决、裁定无关的，自异议裁定送达之日起**15日内**提起诉讼。分两类——

**案外人执行异议之诉**：案外人对执行标的主张所有权或足以排除强制执行的实体权利，异议被驳回后提起，请求停止对该标的的执行（并可一并请求确认权利）。

**申请执行人执行异议之诉**：案外人异议成立、法院裁定中止执行，申请执行人认为案外人无权排除执行，提起诉讼请求对该标的继续执行。

诉讼请求写法（案外人方）：
```
一、请求停止对×××标的（具体财产）的执行；
二、（可选）请求确认×××标的归原告所有/原告对×××标的享有足以排除强制执行的权利。
```
论证重点（须充分）：①原告对执行标的享有的**实体权利性质**（所有权、未过户买受人物权期待权、租赁权、共有权、担保物权等）及其**能否排除强制执行**——这是核心争点，须结合民法典与执行异议复议规定充分论证；②权利取得的时间、合法性、是否善意、是否已支付对价、是否实际占有等要件事实；③与原判决、裁定无关的认定。
法律依据：民诉法第238条；《执行异议和复议案件若干问题的规定》（法释〔2015〕10号，2020修正）第14–28条；解释第305条以下。
附：权属证据、交易凭证、占有/付款证明、执行异议裁定书。

## 通用附件与提交提示
- 本家族文书均须经"核验闸门"完成法条与裁判规则校验后定稿；引用案例须经法律检索后端的案例检索核实。
- 时限红线：撤销仲裁裁决 6 个月；第三人撤销之诉 6 个月；执行异议之诉 15 日——**均为不变期间，逾期丧失救济**，须在动笔前自行核算并复核起算点（提醒用户核对）。
- 份数与格式：按起诉状/申请书标准，正本交受理法院，按对方当事人数备副本。
- 后续衔接：立案后转 `/要件攻防分析`、`/证据目录`、`/庭审提纲`。
