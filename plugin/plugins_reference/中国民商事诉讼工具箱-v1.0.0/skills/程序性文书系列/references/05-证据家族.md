# 02 · 证据家族

> 配合 `00-通用规范.md` 使用。覆盖 5 种文书：①证据保全申请书（含诉前）②调查取证申请书（申请法院调查收集证据）③调查令申请书 ④证人出庭作证申请书 ⑤延期举证申请书（延长举证期限）。
> 法条基准：民诉法 2023 修正（2024-01-01 施行）。现行条号见下方速查表，写入前过核验闸门。

## 现行法条速查（须经核验闸门复核）

| 制度 | 民诉法 2023 | 解释（法释〔2022〕11号）/ 证据规定 |
|------|------------|------|
| 申请法院调查收集证据 | 第67条第2款 | 解释第94–97条（范围/时限/不准许救济）；《民事诉讼证据规定》第20–24条 |
| 证据保全（诉中/诉前） | 第84条 | 解释第98–103条；《证据规定》第25–27条 |
| 举证期限与延期 | 第68条 | 解释第99–101条；《证据规定》第51–54条 |
| 证人出庭作证 | 第75条（作证义务）、第76条、第77条（费用） | 解释第117–120条（举证期限届满前申请、保证书、费用预缴） |
| 鉴定（见 04 庭审家族） | 第79–81条 | 解释第121条 |
| 专家辅助人（见 04 庭审家族） | 第82条 | 解释第122–123条 |

> 关键时限：申请法院调查收集证据应在**举证期限届满前 7 日**提出（《证据规定》）；申请证人出庭、申请鉴定均应在**举证期限届满前**提出（解释第117、121条）。

## ① 证据保全申请书

适用：证据可能灭失或以后难以取得（如易腐物、将被篡改的电子数据、即将离境的证人、待拆除现场）。**诉中**向受理法院、**诉前**向证据所在地/被申请人住所地/有管辖权法院申请。

请求事项写法：
```
请求对……（写明需保全的证据名称、形式、所在地/载体，如某服务器内电子数据、某仓库内货物、某网页内容等）采取……（拍照、录像、封存、提取、公证提取、制作笔录等）方式予以保全。
```
专属要件：①证据有灭失或难以取得的紧迫性；②诉前证据保全应提供担保（法院可责令）；③写明保全方法建议。
法律依据：民诉法第84条；解释第98–103条。
附：证明证据存在与紧迫性的材料、担保材料（诉前）、证据所在地/持有人信息。

## ② 调查取证申请书（申请人民法院调查收集证据）

适用：证据属下列情形之一，当事人及代理人因客观原因不能自行收集——①由国家有关部门保存须法院调取的档案材料；②涉及国家秘密、商业秘密或个人隐私的材料；③确因客观原因不能自行收集的其他材料。

请求事项写法：
```
请求人民法院依法向……（持证单位/个人）调查收集……（证据名称、内容、待证事实），用以证明……。
```
专属要件：①写明证据持有人、证据内容与**待证事实的关联性**；②说明无法自行收集的客观原因（已申请单位拒绝/涉密/档案须法院调取）；③在举证期限届满前 7 日提出。
法律依据：民诉法第67条第2款；解释第94–97条；《民事诉讼证据规定》第20–24条。
附：已自行调查受阻的证明（如单位拒绝函）、证据线索。
> 法院不准许的，当事人可在收到通知后 3 日内书面申请复议一次。

## ③ 调查令申请书

适用：在法院调查取证之外，申请法院签发**调查令**，由委托律师持令自行向有关单位/个人调取与待证事实有关的证据（银行流水、工商档案、不动产登记、通讯记录等）。注意：调查令制度无全国统一立法，依据各地高院规定（如沪、粤、苏、浙等），办理前须确认受理法院所在地是否实行及具体要求。

请求事项写法：
```
请求人民法院签发调查令，准许本案委托诉讼代理人×××（律师执业证号：……，律所：……）持调查令向……（被调查单位）调取……（证据名称、内容），用以证明……。
```
专属要件：①申请人必须委托律师，载明律师与律所信息；②写明被调查单位、调取内容、待证事实关联性；③说明该证据属律师自行调取受阻、需以法院名义调取的情形。
法律依据：民诉法第67条；《民事诉讼证据规定》相关规定；受理法院所在地高院调查令实施细则（**须按地核验**）。
附：律师执业证、所函、调查事项清单。

## ④ 证人出庭作证申请书

适用：申请通知知悉案件情况的单位或个人出庭作证。须经法院通知，未经通知证人不得出庭（双方同意并经法院准许的除外）。

请求事项写法：
```
请求通知证人×××（与当事人关系、知悉事实的来源）出庭作证，证明……（拟证明的具体事实）。
```
专属要件：①在**举证期限届满前**提出；②写明证人身份、与案件关系、拟证明事实；③预缴证人出庭必要费用（交通、住宿、误工等，法院准许后预缴）。
法律依据：民诉法第75条、第76条、第77条；解释第117–120条。
附：证人身份信息、联系方式。

## ⑤ 延期举证申请书（延长举证期限）

适用：当事人在举证期限内提供证据确有困难，请求延长举证期限。

请求事项写法：
```
本案举证期限为××××年××月××日届满，因……（具体困难，如证据须法院调取、鉴定未完成、境外证据公证认证未办结等），请求延长举证期限至××××年××月××日（或延长×日）。
```
专属要件：①在**举证期限届满前**提出；②写明确有困难的具体客观事由；③可一并说明已采取的取证措施与预计完成时间。
法律依据：民诉法第68条；解释第100条；《民事诉讼证据规定》第51–54条。
附：佐证延期事由的材料（鉴定受理回执、调取证据申请、境外公证办理进度等）。
> 法院准许延长的，延长的期限适用于其他当事人。延期申请获准与否影响逾期举证后果，紧急时同步联系承办法官。

## 通用附件与提交提示
- 份数：正本交法院，按对方当事人数备副本。
- 时限红线：调查取证/证据保全/证人出庭/鉴定/延期举证均与**举证期限**强关联，提交前务必核算举证期限届满日（自行计算并提醒用户核对）。
- 后续衔接：取得证据后转 `/质证意见` 审查应对；证据保全/调查取证完成后纳入 `/证据目录`。
