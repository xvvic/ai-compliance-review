# 01 · 保全家族

> 配合 `00-通用规范.md` 使用。本家族覆盖 6 种文书：①诉前财产保全 ②诉讼财产保全 ③续行查封/扣押/冻结 ④变更保全标的物 ⑤解除财产保全 ⑥网络查控财产信息申请书。
> 法条基准：民诉法 2023 修正（2024-01-01 施行）。现行条号：诉讼财产保全=第103条、诉前财产保全=第104条、担保解除=第107条。写入前过核验闸门。

## 现行法条速查（须经核验闸门复核）

| 制度 | 现行条文（民诉法 2023） | 配套（法释〔2022〕11 号 / 法释〔2020〕21 号） |
|------|----------------------|------|
| 诉讼财产保全 | 第103条 | 解释第152条（担保书面通知）、第157条 |
| 诉前财产保全 | 第104条（48小时裁定；30日内不起诉/不申请仲裁则解除） | 解释第152、153、154条 |
| 保全范围 | 第105条（限于请求范围或与本案有关财物） | — |
| 保全方法/禁止重复查封冻结 | 第106条（查封、扣押、冻结等；保全后立即通知） | 《查封规定》（法释〔2020〕21号） |
| 被申请人提供担保解除 | 第107条 | 解释第166条 |
| 保全错误赔偿 | 第108条 | — |
| 复议一次（不停止执行） | 第111条 | 解释第171、172条 |
| 解除保全情形 | 第104条第3款/第107条 | 解释第166条（四种应解除情形） |
| 变更保全标的物 | — | 解释第167条（被保全人提供等值担保且利于执行） |
| 续封/续冻期限 | — | 《查封规定》：冻结存款1年、查封扣押动产2年、查封不动产及其他财产权3年；届满前可续，续办期限不超过原期限 |

> 担保数额：诉前财产保全须提供**相当于请求保全数额**的担保（解释第152条）；诉讼保全是否担保及数额由法院依案情决定。可用保险公司保全责任险保函（诉讼财产保全责任保险）替代现金担保。

## ① 诉前财产保全申请书

适用：起诉/申请仲裁前，情况紧急、不立即保全将使合法权益受难以弥补的损害。向**被保全财产所在地、被申请人住所地或对本案有管辖权的法院**申请。

请求事项写法：
```
请求依法查封/扣押/冻结被申请人×××名下的……（财产名称、性质、数量或数额、所在地/账号/不动产权证号/车牌号等），保全金额（标的额）为人民币×××元，期限为……。
```
专属要件：①情况紧急且难以弥补损害；②提供相当于请求保全数额的担保（写明担保财产或保函）；③声明将在保全后 30 日内依法起诉/申请仲裁。
事实和理由：写明债权/请求基础、被申请人转移财产风险、紧急性、担保安排。
法律依据：民诉法第104条、第105条、第106条；解释第152条。
附：担保材料（保函/现金/财产权属证明）、债权证据、财产线索清单。

## ② 诉讼财产保全申请书

适用：诉讼过程中，因对方行为或其他原因可能使判决难以执行或造成损害。向**受理本案的法院**申请。

请求事项写法：
```
（××××）×××民初×号……一案，请求依法查封/扣押/冻结被申请人×××名下的……（财产明细），保全金额为人民币×××元，期限为……。
```
专属要件：①已立案，写明案号与案由；②是否提供担保及数额由法院依案情决定；③保全限于请求范围或与本案有关财物。
法律依据：民诉法第103条、第105条、第106条；解释第157条。
附：起诉材料/受理通知、财产线索清单、担保材料（如需）。

## ③ 续行查封/扣押/冻结申请书

适用：原查封/扣押/冻结期限将届满、案件尚未审结或执行未完毕，需在**期限届满前**申请续行。

请求事项写法：
```
请求对你院（××××）×××号民事裁定查封/扣押/冻结的被申请人×××名下……（财产明细）续行查封/扣押/冻结，续行期限为……。
```
专属要件：①写明原裁定文号与原保全财产、原到期日；②须在届满前提出（实务建议提前 7–15 日）；③续办期限不超过原保全期限。
法律依据：民诉法第103/104条（保全基础）；《查封规定》（法释〔2020〕21号）关于期限与续行的规定。
附：原保全裁定书、原协助执行通知书（如有）、案件进展说明。
> 续封须在原保全期限届满前办理（实务提前 7–15 日），请自行登记并跟踪保全到期日，避免脱保。

## ④ 变更保全标的物申请书

适用：**被保全人**提供其他等值担保财产且有利于执行，请求将保全标的物变更为其提供的财产。

请求事项写法：
```
一、解除你院（××××）×××号裁定对申请人（被保全人）×××名下……（原财产）的保全；
二、对申请人提供的……（其他等值担保财产名称、价值、权属）采取保全措施，期限为……。
```
专属要件：①申请人为被保全人；②替代财产等值且更利于执行（说明流动性/权属清晰/无权利负担）；③诉讼中申请须写明案件当事人与案由。
法律依据：解释第167条。
附：原保全裁定书、替代财产权属与价值证明（评估报告/权证）。

## ⑤ 解除财产保全申请书

适用情形（任一）：①诉前保全后申请人 30 日内未起诉/未申请仲裁；②财产纠纷案件被申请人（被保全人）提供担保；③保全错误、申请人撤回保全申请、起诉或诉讼请求被生效裁判驳回、法院认为应解除的其他情形。

请求事项写法：
```
请求解除你院（××××）×××号裁定对……（财产明细）采取的查封/扣押/冻结措施。
```
专属要件：写明原裁定文号、保全内容、对应的解除事由（按上列情形对号入座）。
法律依据：民诉法第104条第3款、第107条；解释第166条。
附：原保全裁定书；如以担保解除，附担保材料。
> 解除以登记方式实施的保全的，法院应向登记机关发协助执行通知书。

## ⑥ 网络查控财产信息申请书

适用：保全/执行阶段，申请人请求法院通过**网络执行查控系统（总对总/点对点）**查询被申请人（被执行人）名下银行存款、不动产、车辆、股权、网络资金等财产信息，以确定保全或执行标的。**财产线索清单作为本申请书附件。**

请求事项写法：
```
请求依法通过网络执行查控系统查询被申请人×××（统一社会信用代码/身份证号：……）名下的银行账户存款、不动产、车辆、股权及网络资金等财产信息，以便确定保全/执行标的。
```
专属要件：①写明被申请人精确身份标识（企业统一社会信用代码 / 自然人身份证号）以便系统查控；②说明已穷尽自行调查仍不足，需借助法院查控系统；③附《财产线索清单》。
法律依据：民诉法第103/104条（保全）或第二编执行程序相关规定（执行阶段）；最高院关于网络查控/执行事项委托的规定。
附：**《财产线索清单》（由 `/财产线索调查` 技能生成）**、被申请人主体信息核验材料（工商信息/不动产/车辆登记等）。
> 强衔接：先调用 `/财产线索调查` 产出财产线索调查报告与清单，再据此填本申请书附件；保全获准后登记保全措施到期日并及时办理续行，避免脱保。

## 通用附件与提交提示

- 主体材料：申请人身份/营业执照、法定代表人身份证明、授权委托书（见 06 立案家族）。
- 担保材料：现金/财产权属证明/诉讼财产保全责任险保函。
- 证据材料：债权或请求基础证据、财产线索清单。
- 份数：正本交法院，按被申请人数备副本。
- 期限红线：诉前保全 30 日内须起诉/申请仲裁；续封须届满前办理；对裁定不服 5 日内（实务以裁定送达后）申请复议一次。
