# 04 · 庭审程序家族

> 配合 `00-通用规范.md` 使用。覆盖 7 种文书：①不公开审理申请书 ②回避申请书 ③鉴定申请书 ④线下开庭审理申请书 ⑤通知有专门知识的人出庭申请书（专家辅助人）⑥小额诉讼程序异议 ⑦简易程序异议。
> 法条基准：民诉法 2023 修正（已按现行编号核验）。
>
> **归类说明（提示用户）**：原规格曾将"通知有专门知识的人出庭"置于执行家族，但该制度依民诉法第82条、解释第122–123条属**庭审/证据质证**范畴（针对鉴定意见与专业问题），与鉴定相邻，故归入本家族。如需移回执行家族，请告知。

## ① 不公开审理申请书

适用：案件涉及商业秘密、个人隐私等，当事人申请不公开开庭审理（离婚、涉商业秘密案件当事人申请的可以不公开；涉及国家秘密、个人隐私或法律另有规定的依法不公开）。

请求事项写法：
```
请求对本案〔（××××）×××民初×号〕不公开开庭审理。
```
专属要件：①写明不公开的法定/申请事由（商业秘密、个人隐私等）并简要说明；②区分"依法不公开"（国家秘密/个人隐私/法律规定）与"申请可不公开"（离婚、商业秘密）。
法律依据：民诉法第137条。
> 本系列为对外通用范本，**不嵌入任何特定主体的偏好或既有模板**，按个案事由如实填写。

## ② 回避申请书

适用：审判人员（含法官助理、书记员、司法技术人员、翻译、鉴定人、勘验人）存在法定回避情形，申请其回避。

请求事项写法：
```
请求本案审判人员/书记员/鉴定人×××回避。
```
专属要件：①对应第47条情形逐项说明（本案当事人或其近亲属、与本案有利害关系、其他可能影响公正审理的关系，或接受请客送礼/违规会见等）；②**提出时点**：案件开始审理时提出；回避事由在开始审理后才知道的，可在法庭辩论终结前提出；③说明理由并附线索。
法律依据：民诉法第47条、第48条；解释第43–49条。
> 救济：对回避决定不服，可在接到决定时**申请复议一次**，复议期间被申请人不停止参与本案工作（第50条）。

## ③ 鉴定申请书

适用：待证事实需通过鉴定（如笔迹、伤残、工程造价、财务、产品质量、价格评估等）认定，申请法院委托鉴定。

请求事项写法：
```
请求对……（鉴定对象/事项，如某签名的真实性、某工程造价、某伤情等级）进行鉴定。
```
专属要件：①**举证期限届满前**提出；②写明鉴定事项与待证事实的关联性（与待证事实无关或无意义的不予准许）；③可建议鉴定机构，但由双方协商确定，协商不成由法院指定；④预交鉴定费。
法律依据：民诉法第79–81条；解释第121条。
附：送检材料/检材来源说明。

## ④ 线下开庭审理申请书

适用：法院拟采用在线方式开庭，但当事人不同意在线诉讼，申请改为线下（现场）开庭。在线诉讼遵循自愿原则，当事人不同意的不得强制在线。

请求事项写法：
```
本案〔（××××）×××民初×号〕，申请人不同意采用在线方式开庭，请求安排线下（现场）开庭审理。
```
专属要件：①援引自愿原则；②简要说明线下开庭的必要性（核对原件、当事人/证人当庭质证、网络条件不具备等）。
法律依据：民诉法第16条；《人民法院在线诉讼规则》第1–3条。

## ⑤ 通知有专门知识的人出庭申请书（专家辅助人）

适用：申请1至2名具有专门知识的人出庭，代表当事人对鉴定意见进行质证，或就案件事实所涉专业问题提出意见。

请求事项写法：
```
请求准许本案申请人申请的具有专门知识的人×××（专业领域/职称/单位）出庭，就……（鉴定意见/专业问题）发表意见、进行质证。
```
专属要件：①**举证期限届满前**提出；②人数1–2名；③说明其专业领域与拟发表意见的事项；④相关费用由提出申请的当事人负担；⑤其法庭意见视为当事人陈述。
法律依据：民诉法第82条；解释第122–123条。

## ⑥ 小额诉讼程序异议

适用：当事人认为案件适用小额诉讼程序（一审终审）违反法律规定，提出异议请求转为简易程序其他规定或普通程序。

请求事项写法：
```
本案〔案号〕被适用小额诉讼程序审理，申请人认为本案……（属人身关系/财产确权、需鉴定、已提反诉、标的额超标等排除情形，或不符合适用条件），违反法律规定，请求不再适用小额诉讼程序，转为简易程序的其他规定或普通程序审理。
```
专属要件：①对照第165条适用标准与第166条排除情形论证不应适用；②强调小额诉讼一审终审、救济受限，故异议意义重大。
法律依据：民诉法第165条、第166条、第169条。

## ⑦ 简易程序异议

适用：当事人认为案件不属"事实清楚、权利义务关系明确、争议不大"，不应适用简易程序，请求转为普通程序。

请求事项写法：
```
本案〔案号〕被适用简易程序审理，申请人认为本案案情复杂/争议较大/权利义务关系不明确，不宜适用简易程序，请求裁定转为普通程序审理。
```
专属要件：①论证不符合简易程序适用条件；②可一并主张组成合议庭审理。
法律依据：民诉法第160条、第164条、第170条。

## 通用附件与提交提示
- 份数：正本交法院，按对方当事人数备副本。
- 时限红线：回避（开始审理时/辩论终结前）、鉴定与专家辅助人（举证期限届满前）；程序异议宜在收到适用程序通知后尽早提出。
- 后续衔接：鉴定启动后注意审限/举证期可能顺延（自行核算并提醒用户核对）；程序转换影响审级与审限，同步更新案件管家。
