# 08 · 管辖权异议家族

> 本家族在本系列内**完整实现**，不再转交外部技能。涵盖管辖权异议申请书的全流程起草：材料解析 → 当事人核验 → 管辖规则分析 → 法条校验 → 撰写 → Word 输出。

## 核心原则

- **直奔主题，简洁明了。** 聚焦核心异议理由，不做案件背景铺垫与理论展开，用最短篇幅讲清问题。
- **只引法条，不引案例。** 管辖权异议申请书全文不出现任何案例引用，正文与附件均不列案例。
- **不需要附件。** 正文即完整文书，无需附录法律依据、类案参考等附件。
- **法条须现行有效且经核验。** 条文号以民诉法 2023 修正（2024-01-01 施行）/ 民诉法解释 法释〔2022〕11 号为准，写入前按 `00-通用规范.md` 的核验闸门用法律检索后端（元典/北大法宝，运行时自动适配+灾备，见 qoder.md「外部能力后端」）复核，失败标注`[L4-法条待验证]`。

## 现行法条速查表（民诉法 2023 / 民诉法解释 2022，须核验闸门复核）

| 要点 | 现行依据 |
|------|----------|
| 提出时点 | 民诉法第 130 条第 1 款：当事人对管辖权有异议的，应当在**提交答辩状期间**提出，由人民法院审查 |
| 未异议视为有管辖权（应诉管辖） | 民诉法第 130 条第 2 款：当事人未提出管辖异议，并应诉答辩或提起反诉的，视为受诉法院有管辖权，但违反级别管辖和专属管辖规定的除外 |
| 级别管辖 | 民诉法第 18 条（基层法院：一审一般案件）、第 19 条（中院：重大涉外、本辖区有重大影响、最高院确定）、第 20 条（高院：本辖区有重大影响）、**第 21 条（最高人民法院：全国有重大影响/认为应由本院审理）**；中院一审标的额标准见各高院按法发〔2021〕27 号确定的辖区数额 |
| 一般地域管辖 | 民诉法第 22 条（被告住所地/经常居住地）、第 23 条（特定情形原告住所地） |
| 特殊地域管辖 | 合同第 24 条、保险第 25 条、票据第 26 条、**公司诉讼第 27 条**（公司设立/确认股东资格/分配利润/解散等→公司住所地）、**运输合同第 28 条**（铁路/公路/水上/航空运输和联合运输→运输始发地、目的地或被告住所地）、侵权第 29 条、运输事故损害赔偿第 30 条、海事/船舶碰撞第 31–32 条、海难救助第 33 条 |
| 专属管辖 | 民诉法第 34 条（不动产纠纷→不动产所在地；港口作业→港口所在地；继承遗产→被继承人死亡时住所地或主要遗产所在地） |
| 协议管辖 | 民诉法第 35 条（书面协议选择被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地等**与争议有实际联系**的地点，不得违反级别与专属管辖）；格式条款另见民法典第 496 条与民诉法解释第 31 条 |
| 选择/移送/指定管辖 | 民诉法第 36 条（共同管辖原告选择）、第 37 条（移送管辖）、第 38 条（指定管辖） |
| 合同履行地认定 | 民诉法解释第 18 条；信息网络买卖合同第 20 条 |
| 侵权行为地认定 | 民诉法解释第 24 条；信息网络侵权第 25 条 |
| 不动产纠纷范围 | 民诉法解释第 28 条（物权确认/分割/相邻关系；农村土地承包经营、房屋租赁、建设工程施工、政策性房屋买卖按不动产确定管辖） |
| 住所地/经常居住地认定 | 民诉法解释第 3 条、第 4 条 |
| 管辖权异议处理与移送 | 民诉法解释第 35 条（协议管辖两个以上法院/级别专属冲突处理）、第 36 条（受诉法院发现无管辖权移送）等 |
| 仲裁协议排除诉讼管辖 | 民诉法第 127 条第 2 项（有书面仲裁协议向法院起诉的，告知向仲裁机构申请）；仲裁法第 5、17、20 条（仲裁协议效力） |

## 子文书：管辖权异议申请书

### 适用情形

被告（异议人）收到起诉状副本后，认为受诉法院对本案无管辖权，请求将案件移送有管辖权的法院，或（存在仲裁条款时）请求驳回起诉。

### 起草流程（六阶段）

**阶段零 · 材料解析**：上传材料为 PDF/图片/Word/PPT 时，先解析为结构化 Markdown，再提取信息。已是文本则跳过。

**阶段一 · 信息提取**：从起诉状提取受理法院与案号、原被告信息、诉讼请求与标的额、事实理由中的管辖连接点（合同签订地/履行地、侵权行为地等）、原告主张案由；从证据册提取合同管辖/仲裁条款、履行地线索、工商住所地线索。信息不全用 AskUserQuestion 补充。

**阶段二 · 当事人核验（工商信息后端（企查查/同类，见 qoder.md「外部能力后端」））**：对每个企业当事人执行"模糊搜索→精确查询→比对替换"三步，**禁止跳过搜索直接查工商信息**：
1. 以起诉状名称模糊搜索，锁定企业全称与统一社会信用代码（多结果结合住所地/法定代表人判断，仍不定则问用户）；
2. 用全称/信用代码精确查询名称、信用代码、法定代表人、住所地、经营状态；
3. 与起诉状逐项比对，以工商信息后端最新数据为准（住所地直接影响管辖连接点），重大出入可在"事实和理由"中指出。工商信息后端不可用时标注 `[L4-主体信息待验证]`。自然人当事人跳过此步，必要时向用户确认住所地。

**阶段三 · 管辖规则分析（核心，配合上方速查表）**：
1. **识别案由类型**——合同（被告住所地/合同履行地）、侵权（侵权行为地/被告住所地）、公司（公司住所地）、不动产（专属）、劳动争议、担保（一并起诉按主合同/单独起诉担保人按担保合同）。
2. **检查专属管辖**——优先于一切且不得协议排除：不动产纠纷→不动产所在地；港口作业→港口所在地；继承→被继承人死亡时住所地或主要遗产所在地。
3. **检查协议管辖**——书面 + 与争议有实际联系的法定五类连接点 + 不违反级别/专属；格式条款未合理提请注意（民法典第 496 条、民诉法解释第 31 条）可主张无效；有效仲裁条款则法院无管辖权。
4. **确定法定连接点**——合同履行地按民诉法解释第 18 条（约定优先；货币给付→接收货币方所在地；不动产→所在地；其他→履行义务方所在地；即时结清→交易行为地；未实际履行且双方住所地均不在约定地→被告住所地）；信息网络买卖按第 20 条；侵权行为地按第 24 条、信息网络侵权按第 25 条。
5. **检查级别管辖**——按标的额与当事人住所地是否跨省级辖区判断（各高院按法发〔2021〕27 号确定辖区数额：一般均在或均不在受诉法院省级辖区→中院管 5 亿元以上；一方不在→中院管 1 亿元以上，具体以受诉地高院现行规定为准）。
6. **形成异议理由**——受诉法院非被告住所地/合同履行地/侵权行为地法院；存在有效管辖协议应由约定法院管辖；存在专属管辖；存在仲裁协议法院无管辖权；级别管辖错误；原告虚列被告制造管辖连接点。

**阶段四 · 法条校验（法律检索后端）**：按分析结果先查上方速查表定位，再用 `LAW.article_detail`（传法规名 fgmc + 条号 ftnum + refer_date）核验条文编号与措辞；需模糊查找用 `LAW.article_search` 或 `LAW.semantic_search`。重点确认民诉法为 2023 修正版、解释为 2022 修正版。校验不可用时对相应条文标注`[L4-法条待验证]`。全程不检索案例。

**阶段五 · 撰写**（结构与格式见下）。

**阶段六 · 核验闸门 + Word 输出**：抽取草稿全部法条编号回法律检索后端逐条复核（不可用则标`[L4-法条待验证]`并附待校验清单）；通过后用 `Word 转换后端`（DOCX.md_to_docx）（preset=legal）按 `00-通用规范.md` 格式生成 Word，保存输出目录并回链。用户明示"跳过核验/草稿即可"时，在文首标注"⚠️ 未经法律核验，请勿直接提交法院"。

### 请求事项写法

- 一般情形：「裁定将本案移送 ×× 人民法院管辖。」（移送法院须确为有管辖权法院）
- 仲裁条款情形：「裁定驳回原告的起诉。」（理由：双方有有效仲裁协议，法院对本案无管辖权）
- 级别管辖错误：「裁定将本案移送有管辖权的 ×× 中级人民法院管辖。」

### 文书结构

```
管辖权异议申请书

异议人（被告）：[工商信息后端核验后准确信息]  住所地/法定代表人/统一社会信用代码
被异议人（原告）：[工商信息后端核验后准确信息]  住所地/法定代表人

请求事项：
  [简洁写明：移送 ×× 法院管辖；或驳回起诉]

事实和理由：
  一、[第一点核心理由，直接论述]
  二、[第二点核心理由，退一步论述（如先主张无管辖权，退一步主张移送）]
  三、结论（援引法条，明确提出请求）

此致
[受理法院名称]
异议人：[名称]（盖章）
[日期]
```

### 撰写要点

- 直奔主题，开篇即点核心理由；层次清晰（第一点主理由，第二点退一步）；法条嵌入行文不单列章节；每点控制在 3–5 句内；全文无案例、无附件。

### 法律依据（择需引用，须核验）

民诉法第 130 条（异议提出与应诉管辖）+ 对应管辖条文（第 18–35 条按案由/级别选取）+ 民诉法解释相关条文（第 3、4、18、20、24、25、28、31、35、36 条等按情形选取）；涉仲裁条款加民诉法第 127 条第 2 项与仲裁法第 5、17、20 条。

### 专属要件 / 时限红线

- 管辖异议**必须在答辩期内**（被告收到起诉状副本之日起 15 日内）提出，逾期视为放弃（违反级别管辖、专属管辖的除外）。务必先自行核算答辩期起算点并提醒用户核对。
- 移送目标法院必须确有管辖权，否则异议请求难获支持。

## 特殊情形处理

- **仲裁条款**：请求改为"驳回原告的起诉"，理由为法院对该纠纷无管辖权（民诉法第 127 条第 2 项）。
- **原告虚列被告**：在异议理由中论述该被告与本案无实际法律关系、不适格，其住所地不能作为管辖连接点。
- **请求权竞合**：原告已在违约与侵权间选定请求权的，按其选择对应的管辖规则确定管辖，管辖异议阶段不得变更案由。
- **格式管辖条款**（电商平台等）：审查是否以合理方式提请对方注意、是否排除对方主要权利（民法典第 496 条、民诉法解释第 31 条），不符合的可主张无效。
- **担保与主合同冲突**：一并起诉按主合同确定管辖；仅起诉担保人按担保合同确定管辖。
