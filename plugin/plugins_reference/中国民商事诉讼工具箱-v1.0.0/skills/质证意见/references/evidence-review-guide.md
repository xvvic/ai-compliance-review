# 证据审查与质证意见制作指南

## 一、分析顺序与写作顺序

### 分析思考顺序
**关联性 → 合法性 → 真实性**

理由：
1. 关联性是证据的首要资格条件，无关联的证据直接排除
2. 合法性是对具有关联性证据的价值判断，排除非法证据
3. 真实性审查难度较大，在审查真实性前先审查合法性

### 实务写作顺序
**真实性 → 合法性 → 关联性（含证明力）**

说明：证明力的判断最终体现为对关联性和证明目的的认可与否。在本指南中，证明力审查统一归入"关联性"维度——即使证据具备真实性和合法性，但如果无法支撑对方的证明目的，应在关联性部分表述为"关联性不足，证明目的不认可"。

## 二、质证意见类型（严格三性框架）

### 1. 三性无法确认（最常用）
适用：未核验原件、无法核实真实性。
表述："三性无法确认，证明目的不认可。"

### 2. 真实性异议
适用：非原件/原物，或与原件不符。
表述："对真实性有异议，合法性、关联性及证明目的不认可。"

### 3. 合法性异议
适用：证据主体、取得方式、程序、形式不符法律规定。
表述："对合法性有异议，真实性、关联性及证明目的不认可。"

### 4. 关联性异议（含证明力不足）
适用：证据与待证事实无关，或虽有关联但不能证明对方主张。
表述："对关联性有异议，证明目的不认可。"
或："对真实性、合法性无异议，但关联性不足，证明目的不认可。该证据仅能证明[事实A]，不能证明[对方主张的事实B]。"

### 5. 无异议
适用：对证据的三性均无争议。
表述："对证据的三性无异议，对证明目的认可。"

## 三、各类证据审查要点

### （一）书证

**复印件**：
- 未提供原件核对→真实性不确认
- 法律依据：《民事诉讼证据规定》第61条、第90条

**单位证明材料**：
- 形式要件（第115条）：单位负责人签章+经办人签章+单位印章，三者缺一不可

**私文书证**：
- 真实性举证责任在主张方（第92条）
- 有制作者签名/盖章/捺印的，推定真实

### （二）物证
- 审查外部特征、物质属性
- 要求出示原物

### （三）视听资料
- 是否完整、有无剪辑、是否原始载体
- 合法性边界：严重侵害他人权益/违反禁止性规定/严重违背公序良俗的方法形成或获取的，不得采信（第106条）

### （四）电子数据
- 是否提供原始存储介质
- 发件人/收件人/聊天对象身份能否确认
- 内容是否完整

**真实性推定条件**（第94条）：
1. 由当事人提交的于己不利的电子证据
2. 中立第三方平台提供或确认的
3. 正常业务活动中形成的
4. 档案管理方式保管的
5. 当事人约定方式保存的

### （五）证人证言
- 是否出庭：无正当理由未出庭的书面证言不得作为定案根据（第68条）
- 与案件有无利害关系
- 感知来源是否直接
- 是否前后矛盾
- 是否与其他证据印证

### （六）鉴定意见
- 鉴定材料是否经质证（第34条）
- 鉴定人资质
- 鉴定程序是否违法
- 重新鉴定条件（第40条）

### （七）当事人陈述
- 限于诉讼中陈述
- 代理人陈述视为当事人陈述
- 自认规则（第92条）

## 四、质证意见写作模板

### 模板一：三性无法确认
```
[证据编号]：[证据名称]
质证意见：三性无法确认，证明目的不认可。
该证据为[复印件/截图/打印件]，未提供原始载体予以核实，三性无法确认。即便该证据为真，亦不能证明[对方主张的事实]。
```

### 模板二：真实性异议（书证复印件）
```
[证据编号]：[证据名称]
质证意见：对真实性有异议，合法性、关联性及证明目的不认可。
该证据为复印件，未提供原件核对。根据《民事诉讼证据规定》第90条，无法与原件核对的复制件不能单独作为认定事实的根据。
```

### 模板三：真实性异议（电子数据）
```
[证据编号]：[证据名称]
质证意见：三性无法确认，证明目的不认可。
该[微信记录/邮件/短信]为截图，未提供原始存储介质核实，无法确认对象身份及内容完整性，真实性无法确认。
```

### 模板四：合法性异议（单位证明）
```
[证据编号]：[证据名称]
质证意见：对合法性有异议，证明目的不认可。
该单位证明仅有单位印章，无负责人及经办人签章，不符合《民诉法解释》第115条形式要件。
```

### 模板五：关联性异议（含证明力不足）
```
[证据编号]：[证据名称]
质证意见：对真实性、合法性无异议，但关联性不足，证明目的不认可。
该证据仅能证明[实际能证明的事实]，不能证明[对方主张的事实]，无法达到其证明目的。
```

### 模板六：证人证言
```
[证据编号]：[证据名称]
质证意见：对合法性有异议，证明目的不认可。
该证人无正当理由未出庭作证，根据《民事诉讼证据规定》第68条，不得作为认定事实的根据。
```

## 五、常用法律依据速查

| 法条 | 内容要点 |
|------|---------|
| 民诉法第71条 | 书证原件、物证原物提交要求 |
| 民诉法解释第104条 | 质证围绕真实性、合法性、关联性 |
| 民诉法解释第106条 | 非法证据排除 |
| 民诉法解释第115条 | 单位证明材料形式要件 |
| 证据规定第61条 | 书证原件提交 |
| 证据规定第68条 | 证人出庭要求 |
| 证据规定第90条 | 复制件不能单独定案 |
| 证据规定第92条 | 私文书证举证责任 |
| 证据规定第94条 | 电子数据真实性推定 |
| 证据规定第34条 | 鉴定材料须经质证 |
| 证据规定第40条 | 重新鉴定条件 |
