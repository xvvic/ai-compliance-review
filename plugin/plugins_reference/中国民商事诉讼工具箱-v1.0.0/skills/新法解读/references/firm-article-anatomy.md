# 头部律所新法解读文章解剖与输出骨架

> 本文档抽取头部律所新法解读文章的**结构共性与写作手法**，作为本技能输出的骨架库。仅提炼结构与方法，不复制任何具体文章内容。

## 一、头部律所文章的共性结构

拆解大量律所"新规速评/新法解读"后，可归纳出稳定的五段式骨架：

1. **背景与速览**——一句话点明立法背景、出台时点与"谁受影响最大"，公众号体常作导语钩子。
2. **新旧对比**——以表格或并列方式呈现条文变化，这是律所文章区别于"法条复读"的核心，对比表是最常用手法。
3. **核心制度解析**——挑 3-5 个真正改变规则的制度逐项拆解（立法意图 + 实务后果 + 风险量化）。
4. **行业/场景落地**——把制度变化翻译成具体行业的"该做什么"，给 Checklist 或流程卡。
5. **结语与声明**——趋势研判 + 免责声明。

## 二、写作手法要点

- **对比驱动**：先有对比，后有观点；观点必须能回指到具体条款。
- **风险量化**：用🔴/🟡/🔵 等显性标签替代模糊的"需注意"，让读者一眼抓重点。
- **动作可执行**：合规建议写成"修订 X 协议第 Y 条""新增 Z 流程"，而非"加强管理"。
- **克制法言法语**：面向业务/客户的解读尤其要把术语翻译成业务语言。
- **来源透明**：引官方解读、答记者问、配套细则时标注出处。

## 三、标题公式

- 对外/公众号：`[新法解读]《XX法》核心变化与实务指引：行业影响与合规清单`
- 内部简报：`《XX法》修订要点速评——对我司{行业}业务的影响与应对`
- 客户培训：`一文读懂《XX法》：{行业}企业必做的 N 件事`

## 四、三档深度输出骨架

### 速览（≤1500 字）
```
# [速览]《XX法》改了什么
> 文号 ｜ 施行日期 ｜ 面向行业
1. 一句话速览
2. 新旧对比表（精简版，只列关键变化）
3. 核心变化 3-5 条（每条一句）
4. 依据清单 + 法律声明
```

### 标准（3000–5000 字）
```
# [新法解读]《XX法》核心变化与实务指引
> 文号 ｜ 施行 ｜ 面向行业 ｜ 视角
一、一句话速览
二、新旧对比表（完整）
三、核心制度深度解析（3-5 项，含风险等级）
四、行业影响与合规清单（业务场景→受影响点→风险等级→动作）
八、依据与来源清单 + 法律声明
```

### 深度（8000 字+）
```
在标准档基础上增加：
五、结合本公司的落地建议（选"是"时，工商信息后端核验）
六、域外对标（如数据/反垄断/合规等可比领域）
七、争议解决预判（举证责任/管辖/裁判口径影响 + 相关案例）
```
