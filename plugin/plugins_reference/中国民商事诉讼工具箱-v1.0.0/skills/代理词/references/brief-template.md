# 代理词格式模板与写作指引

适用于本技能两种模式、三个审级。Word 排版统一见 SKILL.md 第 7 节。

---

## 模板一 · 代理意见（庭前/开庭准备代理词）

```
代理词

尊敬的审判长、审判员：

[律师姓名]系[律所名称]律师，受[委托人全称]委托，担任[原告/被告/上诉人/被上诉人][全称]与[对方全称][案由]纠纷一案（案号：（[年]）[字]第[号]号）的诉讼代理人。现根据本案事实与法律，发表如下代理意见：

一、关于[争议焦点一]
  （总）我方核心观点……
  （破）对方主张……不能成立，理由是……
  （立）事实层面：……（证据×号，第×页）；法律层面：[法条 条/款/项]……
  （结）故……

二、关于[争议焦点二]
  ……

综上，[重申诉讼请求/抗辩主张]，恳请贵院依法予以[支持/驳回对方请求]。

此致
[法院全称]

                                   代理人：[签名]
                                   [律师事务所名称]
                                   [年] 年 [月] 月 [日] 日
```

写作指引：
- 开篇身份与受托关系一句话交代，进入正题；一审可适度铺陈事实，二审/再审从简（法官已阅卷）。
- 每个争议焦点一个"总—破—立—结"单元；事实主张后标证据编号与页码。
- 结论重申诉请/抗辩，不引入正文未论述的新观点。

## 模板二 · 补充代理意见（庭后）

```
补充代理意见

[法院全称]：

关于[原告全称]与[被告全称][案由]纠纷一案（案号：（[年]）[字]第[号]号），[代理人身份]受[委托人名称]委托，根据[年月日]庭审情况，现就庭审中涉及的争议焦点补充代理意见如下：

一、关于[争议焦点一]的补充意见
  （一）庭审中双方观点
      1. 对方主张：……（庭审笔录第×页）
      2. 我方主张：……
  （二）补充论证
      1. 事实层面：……（证据×号，第×页）
      2. 法律层面：[法条 条/款/项]
      3. 证据补强：……
  （三）小结

二、关于对方[新抗辩/新证据]的回应
  （一）对方新主张概述
  （二）我方反驳意见（破）
  （三）我方依据（立）

综上所述，[重申诉讼请求/答辩意见]。恳请贵院依法[判决/裁定]。

此致
[法院全称]

                                   代理人：[签名]
                                   [律师事务所名称]
                                   [年] 年 [月] 月 [日] 日
```

写作指引：
- 引言简明：代理人身份 + 补充性质 + 触发原因（庭审中出现的具体情况）；不重复案件事实。
- 只就新焦点/薄弱环节/对方新抗辩/法官关注点补充；援引前期观点一句话带过。
- 每个观点"事实+证据+法律"三重支撑；引用对方庭审陈述标笔录页码，引用证据标编号。

## 模板三 · 二审代理词要点叠加（在模板一/二基础上）

二审代理词在上述结构上落实"三点"（详见 SKILL.md 第 4 节）：

1. **焦点对齐上诉请求**：开篇或第一部分明确"本案二审争议焦点应为……（紧扣上诉请求与上诉理由）"；对不应成为二审焦点的问题，单列说明"……不应作为二审审理焦点，因为[一审已查明且未上诉/对方无新证据/与上诉请求无关]"。
2. **三段论先因后果**：核心争点按"大前提（法律规范）—小前提（案件事实）—结论"展开；先论责任/行为成立与否（因），再论损害/数额（果）。针对一审败诉的真正落点（败在"因"还是"果"）对症补强。
3. **另查事实+证据页码**：对二审应另行查明的事实专门陈述，每项事实后罗列对应书面证据的编号与页码。

二审身份措辞：我方为上诉人时称"上诉人"，对方为"被上诉人"；被上诉答辩时反之。

## 模板四 · 再审代理意见要点

- 紧扣再审事由：逐项对照民诉法第 211 条所列情形（如有新证据、原判认定事实缺乏证据证明、适用法律错误、违反法定程序等），论证原裁判构成/不构成相应再审情形。
- 事实主张同样附证据编号与页码；对原裁判的具体说理逐点回应。
- 区分再审审查阶段（是否启动再审）与再审审理阶段（实体改判），措辞与请求相应调整。

---

## 通用格式规范

- 标题：小一号宋体加粗居中
- 一级标题：三号黑体；二级标题：小三号楷体加粗
- 正文：小三号仿宋；行距 28 磅固定值
- 页边距：上下 3.7cm，左右 2.8cm；A4
- 页码：底部居中，小五号宋体；落款右对齐
