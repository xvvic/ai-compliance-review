# 企业法务

<!-- [优化:场景] 重写插件描述，明确AI增量价值和适用场景 -->
企业综合法务助手，帮助法务人员高效完成日常法律事务。覆盖法律文书起草、公司决议生成、类案检索与胜率预判、合规审查、企业背景调查和法律风险评估六大高频场景。基于2024新《公司法》和《民法典》框架。

> **免责声明：** 本插件辅助专业工作流程，不替代法律意见。所有输出应由执业律师审核后方可用于决策。

## 适用角色

- **企业法务** -- 日常法律文书起草、合规审查、案例研究
- **法务总监** -- 综合法律事务管理、风险评估和决策支持
- **合规经理** -- 业务合规审查、监管政策追踪
- **总经理/CEO** -- 交易对手调查、重大事项风险评估

## Skills

<!-- [优化:知识] 补充每个 Skill 的具体功能描述 -->

| Skill | 说明 |
|-------|------|
| 法律文书 | 输入事实和诉求，自动适配文书类型（律师函/起诉状/答辩状/代理词/法律意见书），生成符合规范的初稿 |
| 公司决议 | 输入决议事项，自动判断决议类型和表决比例，生成股东会/董事会决议及配套章程修正案 |
| 查案例 | 输入案情描述，匹配类似案例裁判规则，预判胜率并提供诉讼策略建议 |
| 风险评估 | 输入业务事项，构建法律风险矩阵（可能性 x 严重度 x 可控性），输出分级应对方案 |
| 合规审查 | 输入业务事项，覆盖个保/广告/资质/数据安全等合规领域，输出整改清单 |
| 企业背景调查 | 输入公司名称，综合分析工商/股权/诉讼/经营数据，输出风险评级报告 |
