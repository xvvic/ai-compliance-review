# /查案例 -- 类案匹配与裁判观点分析

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

根据案情描述进行类案匹配，提炼裁判观点和裁判规则，预判胜率并制定诉讼策略建议。

**重要提示**：本技能辅助类案检索工作流程，不构成法律意见。胜率预判基于已知裁判规则的初步分析，仅供参考。实际诉讼结果受证据质量、法官裁量、地区差异等多种因素影响，建议委托专业诉讼律师。

**独立能力（无需连接器）**

- 案情要素提取与案由定性
- 裁判规则梳理与类案分析
- 多维度胜率预判
- 诉讼/应诉策略建议
- 诉讼成本效益参考
- 执行可行性评估

**增强能力（连接器加持）**

- ~~Notion -> 将类案分析报告写入Notion知识库，便于律师团队协作讨论

## 调用方式

```
/查案例 <案情描述>
```

检索以下案情的类似案例：@$1

接受案情描述（文本）或相关法律文件（起诉状/判决书/合同等）。

## 输入要求

接受材料后，确认以下信息：

1. **案由**：合同纠纷/侵权纠纷/公司纠纷/知识产权/劳动争议/其他——确定检索范围
2. **核心争议焦点**：哪些事实或法律适用存在分歧
3. **当事人立场**：用户是原告/被告/第三方
4. **关键事实**：合同条款、违约行为、损失金额、时间线等
5. **目标**：了解裁判规则/评估胜率/制定策略/准备证据
6. **管辖地**（可选）：案件在哪个省市审理——不同地区裁判口径可能不同

用户只描述大致案情也可继续，对缺失信息做合理假设并标注。

## 执行流程

### 第一步：案情要素提取

从用户描述中提取类案检索关键要素：

- **案由定性**：根据最高院《民事案件案由规定》确定案由类别
- **法律关系**：识别涉及的法律关系（合同/侵权/物权/公司治理等）
- **争议焦点**：提炼核心争议焦点（合同效力/违约认定/损害赔偿/股权确认等）
- **关键事实特征**：提取影响裁判结果的关键事实（金额/期限/行为方式/主观过错等）
- **适用法律**：初步判断适用的法律条文和司法解释
- **诉讼时效检查**：主动计算时效是否可能已届满（普通3年/特殊1年）

### 第二步：类案匹配分析

1. 基于内置裁判规则知识库进行类案分析
2. 标注"[基于内置知识分析，非实时检索]"
3. 建议用户通过中国裁判文书网自行检索补充验证

基于裁判规则进行类案分析：

**裁判规则梳理**：
- 最高院指导性案例中的相关裁判规则
- 最高院司法解释中的明确规定
- 各地高院审判指引中的裁判口径
- 《九民纪要》相关条目（如适用）

**类案要素对比**（对每个匹配的裁判规则）：
- 事实相似度：关键事实是否匹配
- 法律适用相似度：争议焦点是否一致
- 结论参考价值：裁判规则的权威性和普适性

**地区差异提示**：标注已知的地区裁判口径差异（如违约金调整标准、加班费计算基数等）。

### 高频案由专项分析框架

如案由属于以下高频类型，启用专项分析框架：

**劳动争议**：
- 核心争点：劳动关系认定、解除合法性、经济补偿/赔偿金计算
- 关键证据：劳动合同、工资流水、考勤记录、解除通知、送达回执
- 裁判趋势：倾向保护劳动者，举证责任多在用人单位（用人单位需举证解除合法性）
- 特别注意：仲裁前置（时效1年）；未签合同双倍工资时效从入职第二个月起算
- 常见争议：N/N+1/2N适用条件、加班费计算基数、年假折算

**合同纠纷**：
- 核心争点：合同效力、违约认定、损失计算、违约金调整
- 关键证据：合同文本、履行记录、损失凭证、催告函、沟通记录
- 裁判趋势：尊重意思自治，违约金酌情调整至实际损失的130%左右（九民纪要第50条）
- 特别注意：时效3年；约定管辖优先；电子证据的证据能力
- 常见争议：不可抗力vs情势变更、预期违约、定金罚则与违约金并用

**知识产权侵权**：
- 核心争点：权利基础有效性、侵权比对（实质性相似）、赔偿额计算
- 关键证据：权利证书/登记、侵权产品公证保全、销售数据、利润审计
- 裁判趋势：法定赔偿为主（500万以下），知识产权惩罚性赔偿适用趋增
- 特别注意：管辖集中在中级法院或知识产权法院；商标侵权可申请行政查处
- 常见争议：合理使用/在先使用抗辩、赔偿额举证困难

**公司纠纷**：
- 核心争点：股东知情权、利润分配、股东除名、公司僵局
- 关键证据：公司章程、股东会决议/会议记录、财务报告、出资凭证
- 裁判趋势：保护中小股东权益，严格审查排除股东权利的决议
- 特别注意：股东代表诉讼前置程序（书面请求监事会/董事会）；公司僵局的司法解散条件严格

### 第三步：胜率预判

> 胜率评估基于类似案例的裁判趋势推断，实际诉讼结果受多因素影响，仅供参考。

| 评估维度 | 有利（+） | 中性（=） | 不利（-） |
|---------|----------|----------|----------|
| **法律依据** | 有明确法条+司法解释支持 | 法条规定模糊，需解释 | 缺乏法律依据或法条不利 |
| **裁判规则** | 指导性案例/主流裁判观点支持 | 裁判观点不统一 | 主流观点不利 |
| **证据情况** | 关键证据齐备+证据链完整 | 部分证据缺失但可补充 | 关键证据缺失 |
| **程序风险** | 无程序障碍 | 存在可克服的程序问题 | 诉讼时效/主体/管辖有硬伤 |

**证据强度分级**（对用户已有证据逐项评估）：

| 强度等级 | 定义 | 示例 |
|---------|------|------|
| **A（强）** | 直接证明待证事实，难以被推翻 | 签字合同原件、银行转账记录、公证书 |
| **B（中）** | 可证明待证事实，但对方可提出反证 | 微信聊天记录、电子邮件、证人证言 |
| **C（弱）** | 仅能间接证明，需其他证据补强 | 单方制作的文件、口头陈述、间接推定 |

**胜率评级**：
- **高（70%以上）**：多数维度有利，关键证据为A级，无明显不利因素
- **中（40-70%）**：存在争议点但整体倾向有利，或各维度参差
- **低（40%以下）**：多数维度不利，或关键证据缺失
- **不确定**：关键事实不明或法律适用存在重大争议

### 第四步：诉讼成本参考

帮助用户做成本效益决策：

| 费用项目 | 估算方式 |
|---------|---------|
| 诉讼费 | 财产案件：按标的额阶梯计算（1万以下50元，1-10万2.5%，10-20万2%...） |
| 保全费 | 5000元以下免费，5000-10万30元，10万以上0.5%+520元（上限5000元） |
| 律师费 | 按标的额、案件复杂程度和地区标准估算范围 |
| 时间成本 | 一审普通程序6个月，简易程序3个月，二审3个月 |

**成本效益提示**：当预期收益（胜诉金额 x 胜率 x 执行到位率）低于诉讼总成本时，建议优先考虑协商/调解。

### 第五步：执行可行性评估

胜诉不等于实际拿到钱。对执行风险做初步评估：

| 评估项 | 说明 |
|--------|------|
| 对方偿付能力 | 基于已知信息判断对方是否有能力履行判决 |
| 可供执行财产线索 | 不动产/车辆/银行账户/股权/应收账款 |
| 执行难度 | 是否可能转移财产、是否有其他债权人 |
| 保全建议 | 是否建议在诉前/诉中申请财产保全 |

### 第六步：策略建议

**如用户为原告：**
- 诉请策略：诉讼请求是否需要调整（增加/减少/变更）
- 证据准备：需补充收集的证据、保全建议
- 和解评估：是否建议诉前/诉中调解，合理和解区间

**如用户为被告：**
- 抗辩策略：可用的抗辩理由和反诉可能
- 程序抗辩：诉讼时效/管辖权/主体资格抗辩
- 风险控制：最大可能败诉金额、是否需要提前做财产保全应对

**通用建议**：
- 证据保全的紧急程度
- 是否建议先行保全或先予执行
- 替代争议解决方式评估（调解/仲裁/诉讼的利弊）

## 质量标准

- 案由定性必须参照最高院《民事案件案由规定》分类
- 裁判规则引用须标明来源和权威等级
- 胜率预判须明确说明评估的局限性，不得做确定性承诺
- 不得虚构不存在的案例编号或裁判文书编号
- 已知的地区裁判口径差异必须标注
- 证据分析必须区分用户已有证据和建议补充证据，并评估强度等级
- 诉讼时效必须主动检查和提示
- 策略建议必须考虑对方可能的抗辩点
- 诉讼成本参考必须基于案件具体情况估算
- 执行可行性不可遗漏（胜诉不等于回款）

## 关联技能

- **法律文书** -- 根据策略建议起草起诉状/答辩状/代理词
- **风险评估** -- 对诉讼结果做更全面的风险评估
- **企业背景调查** -- 调查诉讼对手的财产状况和信用情况
